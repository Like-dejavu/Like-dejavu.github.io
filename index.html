类与对象

类是没有实体的模板、设计图，它定义了某一类事物的共同特征（属性）和行为（方法）。

以“手机”为例，它作为一个类，包含以下核心要素：

  • 共同特征（属性）：品牌、型号、颜色、内存
  • 共同行为（方法）：打电话、发消息、拍照

类本身是抽象的，无法直接看到、摸到，其核心作用是约定这类事物必须具备的属性和行为。

对象是根据类的模板创建出来的具体实体，是类的实例化结果。

类与对象的核心关系是：
类是对象的模板，对象是类的实例。
一个类可以创建无数个对象，所有对象都拥有类定义的属性和行为，但不同对象的属性值可以存在差异。

一、定义类（模板）

类的定义格式：
修饰符 + class + 类名 + { 属性 + 方法 }

public class Phone {
    // 成员变量（属性）
    String brand;  // 品牌
    String model;  // 型号
    String color;  // 颜色
    int memory;    // 内存

    // 成员方法（行为）
    public void call() {
        System.out.println(brand + " " + model + " 正在打电话");
    }

    public void sendMessage() {
        System.out.println(brand + " " + model + " 正在发消息");
    }
}

Java 格式化当前系统时间

在 Java 开发中，我们经常需要获取当前系统时间，并按照指定格式（如 yyyy-MM-dd HH:mm:ss）进行格式化输出。下面是一个完整、可直接在 IDEA 中运行的示例代码，同时附上详细说明。

完整可运行代码

import java.text.SimpleDateFormat;
import java.util.Date;

public class Main {
    public static void main(String[] args) {
        // 1. 创建SimpleDateFormat格式化对象
        SimpleDateFormat sdf = new SimpleDateFormat();
        // 2. 设置自定义时间格式：yyyy-MM-dd HH:mm:ss a（a代表上午/下午）
        sdf.applyPattern("yyyy-MM-dd HH:mm:ss a");
        // 3. 获取当前系统时间
        Date date = new Date();
        // 4. 格式化并输出时间
        System.out.println("现在时间：" + sdf.format(date));
    }
}

给定要筛数值的范围 n ，从2开始，将每个素数的各倍数标记成合数。如果一个数没有被之前的数标记，那它一定是素数预先建立数组来存储从 1 至 n 的所有素数，方便多次查找

public class SieveOfEratosthenes {
public static void main(String[] args) {
int n = 100; // 查找 1 到 100 的所有素数
boolean[] isPrime = new boolean[n + 1];

    for (int i = 2; i <= n; i++) {
        isPrime[i] = true;
    }

    for (int i = 2; i * i <= n; i++) {
        if (isPrime[i]) {
            for (int j = i * i; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }

    System.out.println("素数列表：");
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]) {
            System.out.print(i + " ");
        }
    }
}

}

埃氏筛的优化：欧拉筛
在埃氏筛法中，一个合数来说可能会被筛多次，比如6可以被2筛去，也可以被3筛去，而欧拉筛要做的事情就是让一个合数只被筛一次。

const int MAXSIZE=1e7; //定义空间大小10^7
bool isPrime[MAXSIZE+1]; //素数表，如果isPrime[i]为true，则i为素数；否则为合数
int primes[MAXSIZE]; //已筛出的素数， 素数会按从小到大的顺序不断加入此数组
int cnt = 0; //已筛出的素数的总数

void Euler_Sieve(int n)
{
memset(isPrime, true, sizeof(isPrime)); // 先将所有的数标记为素数
isPrime[0] = false;
isPrime[1] = false; // 0，1皆不是素数

for (int i = 2; i <= n; ++i)
{                                                    // 遍历2到n间的所有整数
    if (isPrime[i])
        primes[cnt++] = i;                        // i是素数，那么将i添加到已筛出的素数中，cnt++
    for (int j = 0; j < cnt && i * primes[j] <= n; ++j)
    {                                                        // 枚举所有已筛出的质数
        isPrime[i * primes[j]] = false;                       // 标记i的素数倍数为合数
        if (i % primes[j] == 0)
            break;                             // 如果i可以被素数prime[j]整除，跳出循环
    }
}

}
