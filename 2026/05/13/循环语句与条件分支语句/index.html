循环语句与条件分支语句

一、条件分支语句

分支语句用于根据布尔条件，选择性执行不同代码块，分为if-else、switch两类。

1. if-else 分支

基础格式

// 单if判断
if(布尔条件){
    条件成立执行代码;
}

// if-else 二选一
if(条件){
    成立执行;
}else{
    不成立执行;
}

// 多分支 if-else if-else
if(条件1){
    // 条件1满足
}else if(条件2){
    // 条件2满足
}else{
    // 所有条件都不满足
}
特点
java
运行
// 1. 括号内必须是true/false布尔表达式；
// 2. 只有一行代码时大括号可省略，但不推荐；
// 3. 支持嵌套 if，可实现多层判断。

示例
java
运行
int score = 85;
if(score >= 90){
    System.out.println("优秀");
}else if(score >= 60){
    System.out.println("及格");
}else{
    System.out.println("不及格");
}

2. switch 分支
语法格式
java
运行
switch(表达式){
    case 值1:
        执行代码;
        break;
    case 值2:
        执行代码;
        break;
    default:
        所有case不匹配时执行;
}

支持数据类型
java
运行
// byte、short、int、char、String、枚举（JDK7+ 支持字符串）

关键规则
java
运行
// 1. break：跳出 switch，不加会发生case穿透，继续执行后续 case；
// 2. default：可选，匹配不到任何 case 时执行，位置不限；
// 3. case 后只能写常量，不能写变量、区间表达式。

穿透案例（利用穿透实现多值匹配）
java
运行
int month = 2;
switch (month){
    case 1:
    case 2:
    case 12:
        System.out.println("冬季");
        break;
    case 3:
    case 4:
    case 5:
        System.out.println("春季");
        break;
    default:
        System.out.println("未知月份");
}

3. 三元运算符（简易分支）
格式
java
运行
// 布尔条件 ? 条件成立值 : 不成立值

示例
java
运行
int a = 10, b = 20;
int max = a > b ? a : b;

限制
java
运行
// 仅适合简单二选一赋值，不能执行多行代码。

二、循环语句
循环用于重复执行一段代码，Java 提供 for、while、do-while 三种循环，搭配 break、continue 控制循环跳转。
1. for 循环（已知循环次数首选）
标准语法
java
运行
for(初始化变量;循环条件;更新变量){
    循环体代码;
}

执行流程
java
运行
// 初始化 → 判断条件 (true 执行循环体) → 更新变量 → 再次判断条件

示例：打印 1~10
java
运行
for(int i = 1; i <= 10; i++){
    System.out.println(i);
}

增强 for 循环（for-each，遍历数组 / 集合）
java
运行
// 无需下标，仅用于读取数据，不能修改容器长度
int[] arr = {1,2,3};
for(int num : arr){
    System.out.println(num);
}

2. while 循环（未知循环次数）
语法
java
运行
while(布尔条件){
    循环体;
    更新条件变量;
}

特点
java
运行
// 先判断，后执行，条件不成立一次都不运行

示例
java
运行
int i = 1;
while(i <= 5){
    System.out.println(i);
    i++;
}

3. do-while 循环
语法
java
运行
do{
    循环体;
}while(条件);

特点
java
运行
// 先执行一次，再判断条件，最少执行 1 次

示例
java
运行
int i = 10;
do{
    System.out.println("至少执行一次");
}while(i < 0);

三、循环控制关键字
1. break
java
运行
// 直接终止当前整个循环，跳出循环体
for(int i=1;i<=10;i++){
    if(i == 5){
        break; // i=5直接结束循环
    }
    System.out.println(i);
}

2. continue
java
运行
// 终止本次循环，直接进入下一轮循环，不跳出整体循环
for(int i=1;i<=5;i++){
    if(i == 3){
        continue; // i=3跳过本次输出
    }
    System.out.println(i);
}

四、三种循环对比总结
java
运行
// for：明确循环次数，适合数组遍历、固定次数循环；
// while：不确定循环次数，先判断后执行；
// do-while：必须至少执行一次的场景。

五、常见易错点
java
运行
// 1. while/do-while 忘记更新循环变量，造成死循环；
// 2. switch 遗漏 break，引发 case 穿透；
// 3. if 后无大括号时，仅第一行代码属于分支内；
// 4. for-each 循环无法获取数组下标，不能增删元素；
// 5. switch case 不能写区间判断，只能等值匹配。
