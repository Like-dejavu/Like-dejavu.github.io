java常见关键字

关键字 含义
abstract 声明抽象类或抽象方法
assert 断言（JDK 1.4 引入）
boolean 布尔类型
break 跳出循环或 switch 语句
byte 8 位有符号整数类型
case switch 语句中的分支
catch 捕获异常
char 16 位 Unicode 字符类型
class 声明类
const 保留字，未使用（Java 中无此功能）
continue 跳过当前循环剩余部分，继续下一次迭代
default switch 中的默认分支；或接口的默认方法
do do-while 循环的开始
double 双精度浮点数类型
else if 条件分支的否定分支
enum 声明枚举类型（JDK 5 引入）
extends 继承类或接口
final 常量、不可继承类、不可重写方法
finally 异常处理中总会执行的代码块
float 单精度浮点数类型
for 循环结构
goto 保留字，未使用
if 条件判断
implements 实现接口
import 导入类或包
instanceof 判断对象是否为某类型实例
int 32 位有符号整数类型
interface 声明接口
long 64 位有符号整数类型
native 声明本地方法（用其他语言实现）
new 创建对象实例
package 定义包
private 私有访问修饰符
protected 受保护访问修饰符
public 公有访问修饰符
return 方法返回值
short 16 位有符号整数类型
static 静态成员（类级别）
strictfp 严格浮点计算（JDK 1.2 引入）
super 父类引用
switch 多分支选择结构
synchronized 同步锁
this 当前对象引用
throw 手动抛出异常
throws 声明方法可能抛出的异常
transient 序列化时忽略字段
try 开始异常捕获或资源管理块
void 表示无返回值
volatile 保证变量可见性和有序性
while 循环结构
