Java数组详解

一、数组核心概念

Java 数组是固定长度、存储相同类型数据的容器，属于引用数据类型，内存空间连续，依靠索引访问元素。

1.1 核心特性

  • 数组创建后长度固定，无法更改
  • 只能存放同一种数据类型
  • 索引从0起始，末尾索引 = 数组长度 - 1
  • 元素数据类型统一：只能存同一种基本类型或引用类型。

1.2 适用场景

  • 存储数量固定的同类数据
  • 需要下标快速查询数据
  • 作为排序、查找算法基础容器

二、数组声明与初始化

2.1 数组声明

// 推荐写法
int[] score;
String[] name;

// 旧式写法，不推荐
int age[];
