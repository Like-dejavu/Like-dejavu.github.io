Java Swing

一、Swing 基础概念

// Swing 是Java图形界面工具包，用于开发桌面窗口程序
// 全部组件位于 javax.swing 包
// 优点：跨平台、组件丰富、轻量级组件，不依赖本地系统控件
// 顶层容器：JFrame(主窗口)、JDialog(弹窗)、JApplet(网页小程序，淘汰)
1. 核心顶层容器 JFrame
java
运行
import javax.swing.*;

public class TestFrame {
    public static void main(String[] args) {
        // 创建主窗口对象
        JFrame frame = new JFrame("我的窗口");

        // 窗口基础设置
        frame.setSize(400,300);        // 设置宽高
        frame.setLocationRelativeTo(null); // 窗口居中
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); // 关闭窗口退出程序
        frame.setVisible(true);        // 显示窗口
    }
}
java
运行
// 常用关闭参数
// EXIT_ON_CLOSE：关闭窗口直接退出程序
// HIDE_ON_CLOSE：仅隐藏窗口，程序后台运行
// DISPOSE_ON_CLOSE：销毁窗口资源

三、面板 JPanel（承载组件的容器）
窗口不能直接放组件，必须通过 JPanel 中转
java
运行
JPanel panel = new JPanel();
panel.setLayout(null); // 关闭默认流式布局，使用绝对坐标

// 设置组件坐标与大小

l.setBounds(50,50,100,30);
panel.add(l); // 组件添加到面板

frame.add(panel); // 面板添加到窗口
四、三种常用布局管理器
java
运行
// 1. FlowLayout 流式布局（JPanel默认）
// 组件从左到右自动排列，自动换行
panel.setLayout(new FlowLayout(FlowLayout.CENTER));

// 2. BorderLayout 边界布局（JFrame默认）
// 分为东、南、西、北、中五个区域
frame.add(btn, BorderLayout.NORTH);

// 3. null 绝对布局，手动设置bounds坐标
panel.setLayout(null);

七、完整简易登录窗口示例
java
运行
import javax.swing.*;

public class LoginDemo {
    public static void main(String[] args) {
        JFrame frame = new JFrame("登录界面");
        frame.setSize(320,220);
        frame.setLayout(null);
        frame.setLocationRelativeTo(null);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        // 账号
        JLabel userLab = new JLabel("账号：");
        userLab.setBounds(60,40,60,25);
        JTextField userInput = new JTextField();
        userInput.setBounds(120,40,120,25);

        // 密码
        JLabel pwdLab = new JLabel("密码：");
        pwdLab.setBounds(60,80,60,25);
        JPasswordField pwdInput = new JPasswordField();
        pwdInput.setBounds(120,80,120,25);

        // 登录按钮
        JButton loginBtn = new JButton("登录");
        loginBtn.setBounds(100,130,100,30);
        loginBtn.addActionListener(e->{
            String user = userInput.getText();
            String pwd = new String(pwdInput.getPassword());
            if("admin".equals(user) && "123456".equals(pwd)){
                JOptionPane.showMessageDialog(frame,"登录成功");
            }else{
                JOptionPane.showMessageDialog(frame,"账号或密码错误");
            }
        });

        // 全部组件添加到窗口
        frame.add(userLab);
        frame.add(userInput);
        frame.add(pwdLab);
        frame.add(pwdInput);
        frame.add(loginBtn);

        frame.setVisible(true);
    }
}
八、核心易错点
java
运行
// 1. 组件必须先add到面板/窗口，否则界面不显示
// 2. 使用绝对布局(null)时，必须调用setBounds设置坐标宽高
// 3. 窗口设置完所有组件后，最后执行setVisible(true)
// 4. Swing组件线程不安全，复杂程序推荐SwingUtilities.invokeLater
// 5. JPasswordField获取密码不能直接toString，要用new String(getPassword())
